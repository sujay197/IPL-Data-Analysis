# IPL Data Analysis Project

## Overview
This project performs Exploratory Data Analysis (EDA) on IPL datasets using Python.

The analysis includes:
- Most successful teams
- Toss impact analysis
- Top run scorers
- Top wicket takers
- Players with most sixes
- Venue analysis
- Season-wise match trends

## Technologies Used
- Python
- Pandas
- Matplotlib
- Seaborn

## Key Insights
- Mumbai Indians and Chennai Super Kings are among the most successful teams.
- Winning the toss provides a slight advantage.
- Suresh Raina and Virat Kohli are among the top scorers.
- Lasith Malinga leads wicket-taking statistics.
- Chris Gayle dominates six-hitting records.
